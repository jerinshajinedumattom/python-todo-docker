# 📝 Python To-Do App (Dockerized)

## 📌 Project Overview

This is a simple To-Do web application built using Python and Flask. The main idea behind this project is to understand how a basic application can be containerized using Docker and managed with Docker Compose.

The app allows users to add tasks, view them, delete them, and mark them as completed. It runs inside a Docker container, making it easy to set up and run on any system.

---

## ⚙️ How to Run the Project

Follow these steps to run the application on your system:

### 1. Clone the repository

```bash
git clone https://github.com/<your-username>/python-todo-docker.git
cd python-todo-docker
```

### 2. Run using Docker Compose

```bash
docker compose up --build
```

### 3. Open in browser

Go to:
http://localhost:5000

---

## 🐳 Docker Info

* The app is containerized using Docker
* Docker Compose is used to manage and run the container
* Port 5000 is exposed to access the app in the browser
